## JoyAI-LLM-Inference

JoyAI-LLM-Inference is an open-source initiative dedicated to building high-performance inference infrastructure for large language models. We focus on two core missions:

- **Novel Model Architecture Support** — Providing native, optimized runtime support for custom and emerging model architectures beyond mainstream implementations.
- **Inference Acceleration Research** — Developing and integrating state-of-the-art optimization techniques — including but not limited to advanced quantization, speculative decoding, kernel fusion, and efficient memory management to push the boundaries of serving throughput and latency.

*We believe that the next wave of LLM innovation will come not only from training breakthroughs but also from how efficiently and flexibly we can serve diverse model designs at scale. 
Our work bridges the gap between research-stage architectures and production-grade deployment. 
Contributions, discussions, and collaborations are welcome. If you are working on novel model structures or inference optimization, we'd love to hear from you.*
